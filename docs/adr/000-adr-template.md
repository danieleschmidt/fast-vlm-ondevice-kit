## Usage

Copy this template for new ADRs:

```bash
cp docs/adr/000-adr-template.md docs/adr/XXX-descriptive-title.md
```

## ADR Numbering

Use sequential numbering:
- 001, 002, 003, etc.
- Include leading zeros for proper sorting
- Use descriptive but concise titles
